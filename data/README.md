# Data directory

Place the Social Sphere dataset file here (e.g. `ssma.csv`) if you run the notebook locally.

**In the monorepo**, the file is often at:

`Social-Sphere-Project/submissions/team-members/kola-taiwo/data/ssma.csv`

Copy it into this folder **without** deleting the original, or update the notebook’s `read_csv` path to point to your copy.

**If you cannot publish the dataset**, add `*.csv` to the root `.gitignore` and document how authorized users obtain the file.
